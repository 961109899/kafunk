Creates a bounded mailbox bounded by the specified condition. If the condition return true, the mailbox blocks puts, otherwise accepts them.
put a mb
Signature: a:'a -> mb:BoundedMb<'a> -> Async<unit>
Type parameters: 'a

Puts a message into the mailbox. If the mailbox is full, wait until it has room available. If the mailbox has vacancy, add the message to the buffer and return immediately.
