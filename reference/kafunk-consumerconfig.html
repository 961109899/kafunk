Kafunk

ConsumerConfig

Namespace: Kafunk

Kafka consumer configuration.

Record Fields

Record FieldDescription
assignmentStrategies
Signature: (AssignmentStrategyName * AssignmentStrategy) []

The consumer group assignment strategies to use. The group coordinator ensures that all members support the same strategy. When multiple stratgies are supported by all members, the first one in the list is selected. Default: [ "range", ConsumerGroupProtocol.AssignmentStratgies.Range ]
endOfTopicPollPolicy
Signature: RetryPolicy

The poll policy to employ when the end of the topic is reached. Default: RetryPolicy.constantMs 10000
fetchBufferSize
Signature: int

The size of the per-partition fetch buffer in terms of message set count. When at capacity, fetching stops until the buffer is drained. Default: 1
fetchMaxBytes
Signature: MaxBytes

The maximum bytes to return as part of a partition for a fetch request. Default: 1048576
fetchMaxWaitMs
Signature: MaxWaitTime

The maximum time to wait for a fetch request to return sufficient data. Default: 0
fetchMinBytes
Signature: MinBytes

The minimum bytes to buffer server side for a fetch request. 0 to return immediately. Default: 0
groupId
Signature: GroupId

The consumer group id shared by consumers in the group.
heartbeatFrequency
Signature: int32

The number of times to send heartbeats within a session timeout period. Default: 5
initialFetchTime
Signature: Time

The time of offsets to fetch if no offsets are stored (usually for a new group). Default: Time.EarliestOffset
offsetRetentionTime
Signature: RetentionTime

Offset retention time. Default: -1L
outOfRangeAction
Signature: ConsumerOffsetOutOfRangeAction

The action to take when a consumer attempts to fetch an out of range offset. Default: HaltConsumer
rebalanceTimeout
Signature: RebalanceTimeout

The time during which a consumer must rejoin a group after a rebalance. If the consumer doesn't rejoin within this time, it will be ejected. Supported in v0.10.1. Default: 10000
sessionTimeout
Signature: SessionTimeout

The session timeout period, in milliseconds, such that if no heartbeats are received within the period, a consumer is ejected from the consumer group. Default: 10000
topic
Signature: TopicName

The topic to consume.
