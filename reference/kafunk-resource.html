Resource<'r>

Recoverable resource supporting the creation recoverable operations. - create - used to create the resource initially and upon recovery. Overlapped inocations of this function are queued and given the instance being created when creation is complete. - handle - called when an exception is raised by an resource-dependent computation created using this resrouce. If this function throws an exception, it is escalated.
