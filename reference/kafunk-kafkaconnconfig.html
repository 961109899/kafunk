Static members

Static memberDescription
create(...)
Signature: (bootstrapServers:Uri list * clientId:ClientId option * tcpConfig:ChanConfig option) -> KafkaConnConfig

Creates a Kafka configuration object given the specified list of broker hosts to bootstrap with. The first host to which a successful connection is established is used for a subsequent metadata request to build a routing table mapping topics and partitions to brokers.
